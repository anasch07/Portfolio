Professional Social Media Network

Description

  • Anas Chaibi | Med Mongi Saidene | Med Ali Zoghlami

  • Dec 2022 - Feb 2023

A Professional Social Media Network :

  • Users can create their own profile
  • Users can create their own posts
  • Users can save posts
  • Users can follow other users
  • Users can like other users posts
  • Users can comment on other users posts
  • Users can search for other users
  • Users can send messages to other users
  • Users can make audio and video calls to other users
  • Users can manage groups
  • Users can manage events and trainings
  • Admin can manage users
  • Admin can check statistics
  • Admin can manage groups (Soon)
  • Admin can manage posts (Soon)

Technology

  • React
  • Node
  • Express
  • MongoDb
  • Redux
  • Socket.io
  • WebRTC
