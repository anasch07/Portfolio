Professional Social Media Network

Description

  • Anas Chaibi | Med Mongi Saidene | Med Ali Zoghlami

  • Dec 2022 - Feb 2023

A Professional Social Media Network :

  • User can login
  • User can register
  • User can create their own profile
  • User can create their own posts
  • User can save posts
  • User can follow other Users
  • User can like other Users posts
  • User can comment on other Users posts
  • User can search for other Users
  • User can send messages to other Users
  • User can make audio and video calls to other Users
  • User can manage groups
  • User can manage events and trainings
  • Admin can manage users
  • Admin can check statistics
  • Admin can manage groups (Soon)
  • Admin can manage posts (Soon)

Technology
